
# Atualizar nome do Dashboard

## Visão Geral

A variável de ambiente `X_CUSTOM_SHOULD_UPDATE_DASHBOARD_NAME` configura se deve atualizar o nome do dashboard para o valor pré-definido na variável `X_CUSTOM_DASHBOARD_NAME`. Quando ativada, o nome do dashboard será atualizado conforme configurado.

## Configuração

```bash
# .env
X_CUSTOM_SHOULD_UPDATE_DASHBOARD_NAME=true
X_CUSTOM_DASHBOARD_NAME=Painel Whatsapp
```

Tudo que for 'Evolution Manager' será substituído pelo nome configurado na variável `X_CUSTOM_DASHBOARD_NAME`.
